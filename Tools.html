Name Description Platform Availability Rationale
Gimp Gimp is an open source image manipulation software. Available on MacOS, Windows and Linux distributions. Gimp was chosen as the Image editor of choice. This was due to its availability and simplicity.
OBS Studio OBS Studio is a free video recording software. Available on MacOS, Windows and Linux distributions. OBS was chosen to record the animated video presentation because it is a program which powerful, simple and cost-effective.
https://www.vyond.com This application allows a user to create animated videos. Web based application available on Google Chrome, Mozilla Firefox and Microsoft Edge. vyond.com is an easy to use animation tool, to create a presentation that was clear, concise and entertaining in an interesting animated format with no previous experience or skills in doing so.
Microsoft Word Program for creating and editing text documents. Available on Windows, Mac and Web. Microsoft Word was a no brainer. It is readily available, stable and used by almost everyone who relies heavily writing
Microsoft PowerPoint This program is used to create slide presentations Available on Mac and Windows PowerPoint was chosen as an alternative to develop a short example to test how our mobile application will function, look and feel.
Microsoft Teams This is a program which creates an environment used for collaboration. Available on MacOS, Windows and Linux distributions. Microsoft team was agreed on by all team member as our collaboration platform. It has all the features required for remote collaboration. But still simple for new users to adapt and stable.
Citethisforme.com This website allows a user to create their bibliography in many referencing styles. Plagiarism checking is also available. Web based This website was chosen to aid in the construction of our references. It has a free option, simplifies the process allowing more time for development and planning.
Grammarly This application help to identify and correct spelling and grammatical errors. It also has a feature to check for plagiarism Web application, desktop application for mac and Windows. Plugins available for Microsoft Office products Grammarly was selected due to its stability, simplicity and accuracy in checking for grammatical errors and poor writing structure.

Citrix Citrix is a DaaS, SaaS company that aids in desktop virtualisation and remote access. Available on MacOS, Windows and Linux. Citrix would be utilised for server virtualisation, remote access and possibly some cloud based functions.
Norton Anti-Virus Available on MacOS, Windows and Linux. Norton Anti-Virus would be used for virus and hacking protection of the LAN network while our programmers developed ComCare.
